# Contributing Guidelines

Follow the steps below to submit your solution for this task.

---

## 1. Fork the Repository

Click the **Fork** button at the top-right of this repository, or use the link below:

https://github.com/mulearngectcr/fastapi-rate-limting

This will create a copy of the repository under your GitHub account.

---

## 2. Clone Your Fork

```bash
git clone https://github.com/<your-username>/fastapi-rate-limting.git
cd fastapi-authentication-basic
```

---

## 3. Create a Folder with Your Name

Inside the repository, create a folder using your name and place your solution inside it.

You may also include:
- Notes about your implementation
- Sample outputs
- Screenshots (optional)

---

## 4. Commit Your Changes

```bash
git add .
git commit -m "Add submission - Your Name"
```

---

## 5. Push to Your Fork

```bash
git push origin main
```

---

## 6. Create a Pull Request

1. Open your fork on GitHub  
2. Click **Contribute → Open Pull Request**  
3. Submit the pull request to the main repository  

---

## 📝 Pull Request Title Format

Submission - Your Name

Example:

Submission - John Doe

---

## ⚠️ Important Notes

- Create **only one folder with your name**.
- Do **not** modify or delete other submissions.
- Ensure your code runs before submitting the PR.
- Do **not commit `.env` files or API keys**.
- You may include bonus features like caching, ranking, or export options.

---

Happy building 💻
